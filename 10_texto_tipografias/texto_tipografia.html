TEXTO Y TIPOGRAFIA

A los tipos de letra que existen se les denomina fuentes o tipografia

Podemos dividirlo en dos grupos:

  • Tipograficas -> fuentes especificas con nombre (Arial, Time New Roman, Verdana, ...)
  • Genericas -> Fuentes segun sus caracteristicas (wrif, sans-serif, cursive, ...
Cuando elegimos una fuente debemos poner fuentes de reserva 
            
                body{
                    font-family: 'Times New Roman', Times, serif;
                }
Si uan fuente tipografica tiene mas de dos palabras debe ir entre comillas el nombre

Las propiedades para cambiar el estilo de las fuentes/texto son
  • font-family
  • font-size
  • font-weigth
  • font-style

  • font-transform
  • font-aling
  • font-decoration
  • font-ident

  • line-height
  • letter-spacing
