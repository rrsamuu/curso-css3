DISPLAY

Nos permite definir como se comporta una caja

De forma natural existen elementos en linea y elementos de bloque

Los valores de esta propiedad son:

  • inline -> establece que la caja se comporta como si fuera un elemento de linea
  • block -> establece que la caja se comporta como si fuera un elemento de bloque
  • inline-block -> se comporta como un elemento en linea pero acepta width y heigth
  • none -> oculta el elemento pero se sigue renderizando
  • table -> imita el comportamiendo de una tabla
  • list-item -> imita el comportamiento de una lista

  • flex -> maquetar componentes
  • grid -> maquetar layout

PSEUDOELEMENTOS

Se utilizan para dar estilos a partes especificas de un elementos

la sintaxis es 

            
                selector::pseudo-element {estilos}
Existen 5 pseudoelementos
  • ::firt-line (elementos de bloque)
  • ::firt-letter (elemento e bloque)
  • ::before
  • ::after
  • ::selection

PSEUDOCLASES

Son selectores que reaccionan en tiempo real detectando la interaccion del usuario con algunos elementos

Funcionan con todos los selectores de css

Su sintaxis es: 

            
                selector:pseudoclase{
                    estilos
                }

                :pseudoclase{
                    estilos
                }
